# Aadhaar Data Analysis Dashboard

This is a premium, high-performance data analysis dashboard built for the UIDAI Hackathon 2025.

## Features
- **Modern UI**: Built with Tailwind CSS and Glassmorphism design.
- **Interactive Charts**: Powered by Recharts (Pie, Bar, Area charts).
- **Animations**: Smooth transitions using Framer Motion.
- **Responsive**: Works perfectly on Mobile, Tablet, and Desktop.
- **Single File Option**: `index.html` is a standalone version that works without any build step.

## How to Deploy (Live in 1 Minute)

### Option 1: Vercel (Recommended)
1. Go to [Vercel](https://vercel.com).
2. Drag and drop the `index.html` file into the deployment area.
3. Your site is live!

### Option 2: Netlify
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the `index.html` file.
3. Your site is live!

### Option 3: GitHub Pages
1. Create a new repository on GitHub.
2. Upload `index.html`.
3. Go to Settings > Pages and enable GitHub Pages for the `main` branch.

## Local Development
If you want to modify the source code:
1. Ensure you have Node.js installed.
2. Run `npm install`.
3. Run `npm run dev`.
