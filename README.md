# Aadhaar Data Analysis Dashboard

This is a premium, high-performance data analysis dashboard built for the UIDAI Hackathon 2025.

## Features
- **Modern UI**: Built with Tailwind CSS and Glassmorphism design.
- **Interactive Charts**: Powered by Recharts (Pie, Bar, Area charts).
- **Animations**: Smooth transitions using Framer Motion.
- **Responsive**: Works perfectly on Mobile, Tablet, and Desktop.
- **Single File Option**: `index.html` is a standalone version that works without any build step.
